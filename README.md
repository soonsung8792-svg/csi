# 시험정보 카메라 (Android)

시험검사 과정 사진을 찍을 때, 화면 **우측 하단에 시험정보**가 실시간으로 뜨고
그 내용이 **사진에 그대로 새겨져** 저장되는 앱입니다.
사진은 **접수번호별 폴더**로 저장되어, 나중에 같은 접수건을 골라 **이어서 촬영**할 수 있습니다.

## 주요 기능
- 접수건 정보 입력: **접수번호 / CSI 접수번호(2칸 분리)**, 공사명(현장명), 시료명, 비고
- 카메라: **시험단계(시험전/중/후)** 와 **시험항목** 선택 후 촬영
- 촬영 시 우측 하단 시험정보가 사진에 자동 합성되어 갤러리에 저장
- **접수번호별로 저장** → 접수건을 다시 눌러 이어찍기
- **시험항목을 직접 추가/삭제** 가능
- 저장 위치: 휴대폰 갤러리의 `Pictures/시험정보/<접수번호>` 폴더

## APK 만드는 법 (둘 중 하나 선택)

> 이 프로젝트는 "소스코드"입니다. 아래 방법으로 **APK 파일로 변환**해서 폰에 설치합니다.

### 방법 A. GitHub 자동 빌드 — 컴퓨터에 아무것도 설치 안 해도 됨 (추천)
1. https://github.com 에 로그인 (계정 없으면 무료 가입)
2. 오른쪽 위 **＋ → New repository** → 이름 아무거나 → **Create repository**
3. 만든 저장소 화면에서 **Add file → Upload files** 클릭
4. 이 폴더 안의 **모든 파일/폴더를 통째로** 끌어다 올림 → **Commit changes**
   - `.github` 폴더까지 반드시 함께 올려야 자동 빌드가 됩니다.
5. 상단 **Actions** 탭 → 초록/노란 실행 항목 클릭 → 빌드가 끝날 때까지 대기(보통 3~6분)
6. 빌드가 끝나면 그 화면 아래 **Artifacts** 에 `시험정보카메라-apk` 가 생김 → 클릭해 내려받기
7. zip 안의 `app-debug.apk` 를 휴대폰으로 옮겨 설치
   - 설치 시 "출처를 알 수 없는 앱" 허용이 필요합니다.

### 방법 B. Android Studio (컴퓨터에 설치되어 있다면)
1. **Android Studio** 실행 → **Open** → 이 폴더 선택
2. Gradle 동기화가 끝나면 상단 메뉴 **Build → Build App Bundle(s)/APK(s) → Build APK(s)**
3. 완료 알림의 **locate** 클릭 → `app/build/outputs/apk/debug/app-debug.apk`
4. 이 APK를 휴대폰에 옮겨 설치

## 사용 순서
1. 앱 실행 → **＋ 새 접수건**
2. (편함) 상단 **📷 신청서 스캔해서 자동입력** → 시험신청서를 촬영 → 접수번호·CSI·공사명·시료명·비고가 자동으로 채워짐 → **확인하고 필요한 곳만 수정**
   - 직접 타이핑해도 됩니다.
   - 인쇄 글자는 잘 인식되고, 손글씨는 틀릴 수 있으니 꼭 확인하세요.
3. **저장하고 촬영 시작**
4. 위에서 **시험전/중/후** 와 **시험항목** 선택 → 가운데 빨간 버튼으로 촬영
   - 시험항목이 없거나 새 항목이 필요하면 스피너 옆 **＋** 로 즉시 추가할 수 있습니다.
5. 나중에 다시 찍을 땐, 첫 화면에서 **해당 접수건을 눌러** 이어서 촬영
6. 시험항목은 첫 화면 **시험항목 관리** 에서도 추가/삭제 가능 (처음엔 비어 있음)

## 참고 / 사양
- 지원: **Android 10(API 29) 이상**
- 필요한 권한: **카메라** (저장은 별도 권한 없이 갤러리에 기록됨)
- 사진 저장 경로: `Pictures/시험정보/<접수번호>/<접수번호>_<단계>_<항목>_<날짜시간>.jpg`
- 데이터(접수건 목록·시험항목)는 앱 내부에 저장됩니다. 앱을 삭제하면 목록은 지워지지만, **이미 찍은 사진은 갤러리에 그대로 남습니다.**

## 폴더 구조
```
app/src/main/java/com/lab/idcam/
  App.kt              앱 시작 / 데이터 로드
  Store.kt            접수건·시험항목 저장(JSON)
  IdCard.kt           시험정보 그리기(미리보기+사진 공용)
  OverlayView.kt      미리보기 위 실시간 식별표
  MainActivity.kt     접수건 목록 / 이어찍기
  ReceiptActivity.kt  접수건 정보 입력
  CameraActivity.kt   카메라 촬영 + 식별표 합성 + 저장
  GalleryActivity.kt  접수건별 사진 보기
  TestItemsActivity.kt 시험항목 추가/삭제
```
